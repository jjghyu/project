 Аппаратный генератор истинно случайных чисел на ESP32

 Описание
 
Портативное USB-устройство, генерирующее криптографически стойкие 256-битные ключи на основе физического шума.

 Особенности
- Гибридный источник энтропии: RF-шум + тепловой шум чипа
- Архитектура: XOR → фон Нейман → SHA-256
- Выход: готовые 256-битные ключи в HEX-формате
- Скорость: около 4-5 ключей в секунду
- Работает от USB

 Использование
1. Подключить ESP32 к компьютеру
2. Открыть монитор порта (115200 бод)
3. Получать поток криптоключей
